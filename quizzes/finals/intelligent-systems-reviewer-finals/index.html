Activity 6 — How Bagging Works (Order the Steps)

Drag the steps into the correct order (1 → 4).

  1. Final predictions are determined by combining all individual model predictions.
  2. Multiple subsets are created from the original data set with equal tuples, selecting observations with replacement.
  3. A base model is created on each subset.
  4. Each model is learned in parallel and independently.
